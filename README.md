# Walking-Animation
Simple 2D walking animation made in canvas

## How to run 
Install required dependencies.
Then you can run
```html
npm start
```
to run the http-server and go to 
```html
http://127.0.0.1:8080/walk-animation.html
```
to watch the animation in action!
